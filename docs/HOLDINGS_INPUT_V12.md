# v12 持仓输入与万元单位

## 改动

- 全工具金额统一按“万元”输入和展示。
- 当前持仓支持输入基金代码后自动匹配基金名称。
- 当前持仓只需要录入：基金代码、持仓金额（万元）、当前盈亏（万元）。
- 当前盈亏为正表示盈利，为负表示亏损。
- 内部仍按元计算，以保持推荐和调仓逻辑一致。

## CSV 上传格式

```csv
基金代码,持仓金额（万元）,当前盈亏（万元）
000961,18,2
110011,22,-3
```

也兼容旧格式：

```csv
code,name,amount,cost_amount
000961,天弘沪深300ETF联接A,180000,160000
```
