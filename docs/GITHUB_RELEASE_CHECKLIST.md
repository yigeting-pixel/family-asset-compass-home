# GitHub Release Checklist

## Before publishing

- [ ] Replace any private data with sample data
- [ ] Check README
- [ ] Check license
- [ ] Run smoke test
- [ ] Verify Streamlit starts
- [ ] Verify API starts
- [ ] Verify no secrets in repository
- [ ] Confirm compliance disclaimers
- [ ] Add screenshots or demo GIF
- [ ] Create first release tag

## Suggested Repository Name

family-asset-advisor-workbench

## Suggested Description

A family asset allocation advisory workbench for financial advisors, with client profiling, fund portfolio recommendation, suitability checks, rebalancing, approval flow, audit logs, and mini-program API support.
