# 产品说明：家庭资产小管家

## 产品定位

一个面向普通家庭的、友好、私密、本地优先的家庭资产配置评估工具。

它不是销售工具，也不是复杂的顾问后台。它帮助家庭回答：

- 我家的安全垫够不够？
- 未来三年要用的钱应该放在哪里？
- 长期资金可以怎么配置？
- 已经持有的基金要不要调整？
- 如果市场下跌，我家能不能承受？
- 如何和家人解释这套配置？

## 产品语气

- 少术语
- 多解释
- 不恐吓
- 不承诺收益
- 不催促交易
- 尊重家庭隐私

## 核心流程

1. 我的家庭
2. 安全体检
3. 三笔钱规划
4. 基金组合
5. 调整建议
6. 情景模拟
7. 我的记录

## 三笔钱模型

- 安心生活桶：应急金和一年内要用的钱
- 三年目标桶：教育、住房、装修等三年内目标
- 长期成长桶：养老和长期增值
